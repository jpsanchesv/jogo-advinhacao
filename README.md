# 🕹️ Jogo da Advinhação
Esse é um jogo criado com HTML5, CSS3 e Javascript. Um projeto elaborado na minha faculdade de ADS na disciplina de Desenvolvimento Web.

---

## 🚀 Funcionalidades

- 📥 Input de letras para advinhação da palavra
- 🧩 Dica com contexto da palavra 
- 🎮 Validação de tentativas restantes
- ⚠️ Alertas de tentativas acabadas
- ⚠️ Alertas de jogo completo
- 🚫 Não aceita números
- ❌ Letras incorretas

## 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para reforçar conceitos iniciais como:

- Estruturação de páginas com HTML  
- Estilização com CSS
- Lógica de programação
- Manipulação de Query Selector
